For our game Phish Hunt: Spot the Scam, our development plan focuses on creating a simple and realistic phishing‑detection game using Visual Studio Code. We will begin by building the Start screen, which includes the game title, instructions, and a Start button. After that, we will create the first phishing scenario with a message, SAFE/SCAM buttons, a timer, and a score display. Once the basic interaction works, we will add the feedback system that shows whether the player was correct and explains the red flags in the message. We will then add more scenarios and create a loop so the player can move through multiple rounds. Finally, we will build the Game Over screen with the final score and a replay option. Aakash is responsible for researching phishing examples, I (Tasnim) will write the content and build the prototype in VS Code, Abiral will create the visual assets and presentation slides, and Sebastian will test the game. Our main risks are time pressure and coding difficulty, so we will keep the design simple and check progress daily. The goal is to produce a working prototype that clearly teaches players how to spot phishing scams.
