Group Work Responsibility – Phish Hunt Game
Project Overview

Phish Hunt: Spot the Scam is a cybersecurity awareness game designed to help users identify phishing scams and suspicious messages through interactive quiz scenarios.

The game includes:

Welcome page
How To Play popup
20 phishing scenarios
10-second timer for each question
Scoring system
Final score display
